> "복습률 0%는 말이 안 돼요. 즐겨찾기 데이터 이미 있으니 1주일 만에 Bronze 출시하고 사용자 반응 보면서 개선해요!"

### 의사결정 프로세스 하이라이트

**우선순위 공식:**
```
우선순위 = (학습 효과 × 사용 빈도) / 개발 공수
        = (높음 × 매일) / 1주
        = 🔥 최우선!
```

**승인 근거:**
- 복습률 0%는 critical 문제
- 즐겨찾기 데이터 이미 존재 (즉시 활용 가능)
- 간격 반복 학습 75% 효과 검증됨 (연구 기반)
- 재방문율 +20%p 기대

## 👥 Phase 2: 사용자 피드백 (5명 페르소나)

### 평가 점수 요약

| 페르소나 | 점수 | 주요 피드백 |
|---------|------|------------|
| 초보 지수 (24세) | ⭐⭐⭐ (3점) | 처음 사용법이 헷갈림 |
| 직장인 민호 (32세) | ⭐⭐⭐ (3점) | 목표/진도 표시 필요 |
| 완벽주의 수진 (26세) | ⭐⭐⭐ (3점) | 답 비교 기능 필수 |
| 개발자 전제 (32세) | ⭐⭐⭐⭐ (4점) | 진도 표시만 추가하면 완벽 |
| 김상현 (32세) | ⭐⭐⭐⭐ (4점) | Best Answer 신뢰도 중요 |

**평균 점수**: ⭐⭐⭐+ (3.4/5점)

### 공통 장점 (All 5명 동의)

1. **시각적으로 예쁘고 깔끔함** - Toss 스타일 친근한 UI
2. **로딩/에러 처리 훌륭함** - 기본기 탄탄
3. **Best Answer 컨셉 좋음** - 자연스러운 표현 학습

### 공통 불편 사항 (Critical Issues)

#### 🔴 Issue #1: 복습 목표/진도 표시 없음 (5명 모두 언급!)

**문제:**
- "몇 개 해야 하지?" 불안함
- 언제 끝나는지 모름
- 성취감 없음

**사용자 코멘트:**
- 지수: "계속 다음 문장 누르다 보면 언제 끝내야 할지 몰라요"
- 민호: "끝나는 기준이 없어요 (이게 제일 불편!)"
- 수진: "진도 표시 없으면 답답해요"
- 전제: "프로그레스 바 있으면 좋겠어요"
- 상현: "몇 개 남았는지 보고 싶어요"

**제안 솔루션:**
```
시작 시: "오늘 몇 개 연습할까요? [5개] [10개] [20개]"
진행 중: "3/10 복습 중"
완료 시: "🎉 10개 복습 완료!"
```

#### 🔴 Issue #2: 내 답과 Best Answer 비교 불가 (4명 언급)

**문제:**
- 입력한 답이 사라져서 차이점 파악 못 함
- 학습 효과 저하
- "왜 틀렸는지" 모름

**사용자 코멘트:**
- 지수: "제 답이랑 뭐가 달랐는지 모르겠어요"
- 수진: "비교 안 되면 학습 효과 떨어져요"
- 전제: "diff 보여주면 좋겠어요"
- 상현: "차이점 아는 게 제일 중요한데..."

**제안 솔루션:**
```
내 답:  I will write the report.
Best:   I'll write the report.
💡 Tip: 일상 대화에서는 축약형(I'll)이 더 자연스러워요
```

#### 🟡 Issue #3: 복습 주기 기능 미구현 (Bronze MVP 스펙인데 안 보임)

**문제:**
- "오늘/3일/1주" 복습 주기 기능 없음
- 간격 반복 학습 효과 미적용

**사용자 코멘트:**
- 수진: "Spaced Repetition 없으면 장기 기억 안 돼요"
- 상현: "복습 알고리즘이 핵심인데..."

## 📊 우선순위 분석

### 🔴 P0 (긴급 - 사용성 크게 향상)

#### 1. 복습 목표 설정 + 진도 표시

**Impact**: 5명 모두 언급 → 사용성 2배 향상 예상

**구현 범위:**
```typescript
// 시작 시 목표 설정
const [dailyGoal, setDailyGoal] = useState(10)

// 진행 중 프로그레스
<ProgressBar current={3} total={10} />

// 완료 시 축하 메시지
<CompletionModal>
  🎉 10개 복습 완료!
  내일도 힘내요! 💪
</CompletionModal>
```

**예상 효과:**
- 사용자 불안감 해소
- 성취감 증가 → 재방문율 +20%p
- "목표 달성" 동기부여

#### 2. 내 답 vs Best Answer 비교 UI

**Impact**: 4명 언급 → 학습 효과 50% 향상 예상

**구현 범위:**
```typescript
// 정답 확인 후 비교 UI
<AnswerComparison>
  <MyAnswer>{userAnswer}</MyAnswer>
  <BestAnswer>{bestAnswer}</BestAnswer>
  <Tip>{differences}</Tip>
</AnswerComparison>
```

**예상 효과:**
- 학습 효과 증대
- "왜 틀렸는지" 이해 → 반복 오류 방지
- 만족도 3.4점 → 4.5점 예상

### 🟡 P1 (중요 - 학습 효과 증대)

1. **복습 통계** - "오늘 5개 복습, 이번 주 25개"
2. **중복 방지** - 오늘 이미 본 문장 제외
3. **간격 반복 알고리즘** - 오늘/3일/1주 복습 주기

### 🟢 P2 (Nice-to-have)

1. **난이도 조절** - 초급/중급/고급 선택
2. **키보드 단축키 확장** - Space, Esc 등
3. **오프라인 캐싱** - localStorage 활용
